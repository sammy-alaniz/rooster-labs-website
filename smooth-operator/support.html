getting started

Smooth Operator lets you plan DJI waypoint missions visually on a map. Tap or click to place waypoints, configure each one, then export as a KMZ file and upload to DJI Fly for execution.

faq

How do I create a mission?

Tap on the map to place waypoints in order. Each waypoint can be configured with its own height, speed, heading, gimbal pitch, and actions. When you're done, export the mission as a KMZ file.

What drones are supported?

Smooth Operator exports KMZ files that follow the DJI WPML specification, works with the DJI Fly app for execution.

Can I edit an existing mission?

Yes. Import any compatible KMZ file to view and edit its waypoints, then re-export when you're done.

Does the app require an internet connection?

An internet connection is needed to load map tiles. Mission creation and KMZ export work offline.

Does the app collect any data?

No. Smooth Operator does not collect, store, or transmit any personal data. See the privacy policy for details.
